Phonetic Alphabets

These are the phonetic alphabets used particularly by radio operators to spell out words. This script uses the NATO phonetic alphabet.

Usage:

  1. Any word that need to be spelt should be marked with a class "spell" 
    <p>
	Spell this word: <span class="spell">CWSL-WRPG-JR3T-3WWL</span>
</p>
  2. Attach prototype.js javascript framework
  3. Attach the script phonetic.js to the page and you are done.
